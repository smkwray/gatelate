# GateLate — Flight Reliability Explorer

**[Live site](https://smkwray.github.io/gatelate/)**

A static-first dashboard exploring U.S. airline and airport on-time performance using official Bureau of Transportation Statistics (BTS) data. Covers industry-wide trends (1995–present), per-carrier breakdowns (21 carriers), major airport rankings, seasonal patterns, and delay cause analysis.

## What this is

GateLate is a monthly-updating reliability explorer that pulls from two BTS data sources:

- **Summary tables** — 8 official XLSX workbooks covering airline and airport on-time rankings (Tables 1A, 2A, 3–6)
- **TranStats delay cause data** — carrier-level monthly statistics with delay breakdowns by cause

The frontend is a static React app that loads pre-processed JSON. No backend server required.

## Repo layout

- `config/` — BTS page URLs and current example workbook URLs
- `scripts/etl/` — download, inspect, and curate pipeline
- `scripts/` — data validation, reconciliation, and analytical dataset builders
- `apps/web/` — React 19 + Vite 6 + TypeScript frontend with recharts
- `tests/` — pytest suite (data validation, parser edge cases, JSON parity)
- `data/` — raw, intermediate, and processed artifacts (gitignored, regenerated by pipeline)

## Pipeline

1. Download the current BTS summary workbooks:
   - `python scripts/etl/fetch_bts_summary_tables.py`
   - Note: BTS uses Akamai CDN which blocks non-browser clients. Use Playwright for downloads if needed.
2. Inspect workbook sheets:
   - `python scripts/etl/inspect_workbooks.py`
3. Export sheet-level CSVs and manifests:
   - `python scripts/etl/build_curated.py`
4. Build Parquet datasets and export JSON for the frontend:
   - `python scripts/etl/curate_v1_datasets.py`
   - Or: `make curate`
5. Build carrier-level data:
   - `python scripts/etl/curate_carrier_data.py`
6. Build analytical datasets:
   - `python scripts/build_analyses.py`
7. Start the frontend:
   - `pnpm --dir apps/web dev`

## What this is not

- Not a live flight tracker or real-time delay monitor
- Not a booking or recommendation engine
- Not a complete route-level warehouse
